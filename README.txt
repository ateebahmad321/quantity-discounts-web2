Boost average order value with customizable tiered quantity discounts and breaks for WooCommerce, with a real-time live preview.

== Description ==

BulkBoost adds eye-catching quantity-break offers to your WooCommerce product pages, encouraging customers to buy more and increasing your average order value (AOV).

Set up tiered pricing per product, style it to match your brand, and watch every change update instantly in the live preview.


= Features =

* **Quantity breaks per product** — offer "buy more, save more" pricing tiers on any simple product.
* **Fully customizable design** — control the colors, borders, corner radius, and spacing of the offer cards.
* **Typography controls** — set the weight and size of labels, descriptions, and prices, and toggle the strikethrough compare-at price.
* **Real-time live preview** — adjust any setting and see your storefront widget update instantly.
* **Min / Max quantity selector** — let customers choose a quantity within a range, with its own customizable button design.
* **Promotional badge** — highlight an offer with a badge to draw attention.
* **Theme friendly** — designed to fit any WooCommerce-compatible theme.



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bulkboost` directory, or install the plugin through the WordPress Plugins screen.
2. Activate the plugin through the **Plugins** screen in WordPress. WooCommerce must be installed and active.
3. Open **BulkBoost** in the admin menu to style your offer cards.
4. Edit any product, open the **Product data → BulkBoost** tab, and add your quantity tiers.

== Frequently Asked Questions ==

= Does it require WooCommerce? =

Yes. BulkBoost extends WooCommerce, so WooCommerce must be installed and active.

= Will it work with my theme? =

It is built to work with any theme that supports WooCommerce. If you run into a display issue, reach out and we will help.

= How do I set up quantity breaks? =

Edit a product, open the **Product data → BulkBoost** tab, and define your quantity thresholds and prices. You can preview the result before saving.

= Can I change how the offers look? =

Yes. The **BulkBoost → Design Settings** screen lets you customize colors, typography, borders, spacing, and the badge, all with a live preview.
